Keyboard Shortcuts

Main Navigation Shortcuts

Keyboard Shortcut Function
o open bundleList sidebar (left)
SHIFT + o open perspective sidebar (right)
h open hierarchy View
w zoom in 1
s zoom out 1
a move viewport left
d move viewport right
q view all
e zoom in to selected area
TAB preselect next item
SHIFT + TAB preselect previous item
→ preselect next item
← preselect previous item
SHIFT + → add next item to preselection
SHIFT + ← add previous item to preselection
g select all items in marked area
SHIFT + S save current bundle
SHIFT + 1..9 switch to perspective 1..9
1.) Zooming while a boundary is preselected will keep the position of that boundary fixed

Labeling shortcuts

Keyboard Shortcut Function
z undo last action
SHIFT + z redo last action
ENTER add boundary / event marker at marked position
ENTER (when single item is preselected) change label of item
BACKSPACE delete preselected boundary / event marker
SHIFT + BACKSPACE delete selected event(s) or segment(s)
t snap preselected boundary / event marker to nearest boundary / event marker in level above
b snap preselected boundary / event marker to nearest boundary / event marker in level below
x snap preselected boundary / event marker to nearest zero crossing
+ add time to preselected items (expand to right)
- remove time from preselected items (shrink from right)
SHIFT + + add time to preselected items (expand to left)
SHIFT + - remove time from preselected items (shrink from left)
↑ select level above
↓ select level below

Playback shortcuts

Keyboard Shortcut Function
f play entire file
c play selected
SPACE play/pause in view (also use as general stop for playback)
SHIFT + SPACE play/pause in view to end of file with autoscrolling

Correction Tool shortcuts

Keyboard Shortcut Function
1 choose correction tool for first contour
2 choose correction tool for second contour
3 choose correction tool for third contour
4 choose correction tool for fourth contour
5 choose correction tool for fifth contour
0 disable correction tool

Hierarchy Tool shortcuts

Keyboard Shortcut Function
r rotate hierarchy by 90°
n add item before the currently preselected one
m add item after the currently preselected one
SHIFT draw a new link (press and hold)
y delete the currently preselected item
BACKSPACE delete the currently preselected link
SPACE play back the portion defined by the currently preselected item
↓ select next hierarchy path
↑ select previous hierarchy path
